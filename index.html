📋 各代数の定義解説:
  • Dnm / Dburst / Dnormal5 : 悪夢単発ダメージ / 総バーストダメージ / フレーバー変奏5段目本体ダメージ
  • Abase / Patk / Aflat : 基礎攻撃力 / 攻撃力％ / 固定攻撃力加算（実数）
  • Snm : 「悪夢」デバフの1スタック毎秒倍率％

  • Ggear : ギア「ディアボロス」による耐性無視％（12% = 0.12 / 24% = 0.24）
  • totalCD : 悪夢攻撃の固定基礎会心ダメ50% ＋ ラスト・ローズ累積会心ダメバフ%の合算
  • Lvply / Lvenm / IC6 : 自分のレベル / 敵のレベル / 6凸による敵の防御力無視％
  • Stack / Pclear : 悪夢の消費スタック数 / 5段目悪夢清算（バースト）倍率
  • 2.0 : 6凸効果「悪夢ダメージ＋100%」に相当する独立最終乗算バフ倍率
